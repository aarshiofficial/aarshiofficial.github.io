# AARSHI — The Drama Club Website

Official website for **AARSHI**, the Theatrical and Dramatics Society of IISER Kolkata.

## 🎭 Live Site

**https://aarshiofficial.github.io/**

## 📁 Structure

```
AARSHI_WEBSITE/
├── index.html               # Single-page website
├── css/style.css            # All styles (dark + light theme)
├── js/main.js               # Animations, filters, lightbox, theme toggle
├── assets/                  # All images and logos
├── ADMIN_GUIDE.md           # Non-technical editing guide
├── .github/
│   └── workflows/
│       └── deploy.yml       # Auto-deploy to GitHub Pages on push to main
└── README.md
```

## 🚀 Deploying

1. Push to the `main` branch on the `aarshiofficial` GitHub account.
2. GitHub Actions will automatically deploy to GitHub Pages.
3. Enable GitHub Pages: Settings → Pages → Source: GitHub Actions.
4. Live at: **https://aarshiofficial.github.io/**
